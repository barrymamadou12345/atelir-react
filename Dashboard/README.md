# 📊 Dashboard React - Projet Pédagogique

Projet Dashboard moderne pour apprendre React, React Router, Tailwind CSS et React Icons.

---

## 🚀 Démarrage Rapide

```bash
# 1. Installer les dépendances
cd Dashboard
npm install

# 2. Lancer le serveur de développement
npm run dev

# 3. Ouvrir http://localhost:5173
```

## 🎯 Ce que vous allez apprendre

✅ Composants réutilisables (Button, Card)
✅ Props et communication entre composants
✅ useState pour gérer l'état
✅ Événements (onClick, onChange)
✅ map() pour afficher des listes dynamiques
✅ React Router DOM pour la navigation
✅ useLocation pour détecter la page active
✅ Tailwind CSS v4 avec couleurs personnalisées
✅ React Icons (Font Awesome)
✅ Layout responsive (Navbar + Sidebar + Contenu)

## 📁 Structure du Projet

```
Dashboard/
├── src/
│   ├── components/
│   │   ├── layout/          # Navbar, Sidebar, Layout
│   │   └── ui/              # Button, Card
│   ├── pages/               # Home, Page1, Page2, Page3, Settings

│   ├── main.jsx             # Point d'entrée

├── package.json

```
